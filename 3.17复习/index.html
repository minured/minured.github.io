2020.03.17

文件的增删改查

1、文件的查看

  • ~ 表示用户目录
  • . 表示当前目录
  • .. 表示上级目录
  • / 代表根目录
  • pwd 查看当前目录（process word directory 进程目录）
  • ls 显示当前目录的所有文件（ls 路径）
  • ls -1 每行显示一个
  • ls -a 显示所有文件，包括隐藏文件
  • ls -l 查看文件的详细信息，还可以写为 ll
  • cat 显示文件的内容
  • less 分页显示文件内容
  • head 默认显示文件内容的前十行，可以添加参数 -n 8显示文件的前 8 行
  • tail 同理 head 显示内容的后十行 2、文件的增加
  • touch 如果文件不存在，新建一个文件；如果文件存在，则刷新文件的最后更新时间
  • echo 向文件写入内容， > 覆盖写入；>> 追加写入
  • mkdir 创建文件夹，如果文件夹存在，则会失败

不熟练的知识点

  • mv 除了移动文件，还可以重命名文件

  • ls --help 查看帮助，其他命令同理

  • echo $? 查看上一个命令命令执行的结果，一般 0 是成功，1 是失败

  • bash 中，alt + . 输入上次输入的内容

  • 把命令写到一个文件当中，然后sh mybash就可以指执行 mybash，文件中，还可以使用$1、$2、$3 来传参

  • npm cnpm npx yarn 都是包管理工具，用来管理 js 的代码库

  • 使用 git 的步骤，先六条命令配置 git（包括邮箱、用户名、默认编辑器等等），然后 git init 初始化，git add ,git commit, 初次提交绑定远程仓库 git remote origin add xxx；git -u origin master

  • git reflog 查看详细记录

  • git reset –hard xxx 版本切换

  • git branch x 创建 x 分支

  • git checkout x 切换 x 分支

  • git pull 是将远程仓库的改动下载到本地仓库

  • git merge 合并分支

  • ssh-keygen -t rsa -b 4096 就可以在本地创建公钥和私钥，把公钥添加到 github 就可以通过 ssh 访问远程仓库，
