HTML常用标签

2020.03.17

a 标签

作用

anchor 用于超链接，跳转外部页面，跳转内部锚点（#id），跳转邮箱或电话等（mailto:xx@xx.xx;tel:12345678910）

属性

  • href

    1、使用无协议写法href = "//google.com"，协议会继承自当前网页协议。

    2、使用路径的时候，/指的是 http 开启位置的根目录

    3、使用伪协议href = "javascript:;屏蔽 a 标签的默认跳转行为，可以用来触发其他操作

  • target

    1、_blank 新标签打开

    2、iframe 已经很多使用(_top 在最外层窗口打开，_parent 在父窗口打开，_self 在当前窗口打开)

  • rel

    1、用于指定当前文档与被链接文档之间的关系

    2、rel = noopener：但 a 标签使用target = _blank时，新标签可以通过window.opener访问原标签，原标签可能会被改变；使用了rel = noopener就可以阻止window.opener，不能进行跨窗口访问。

img 标签

发出 get 请求，显示一张图片

属性

  • alt 图片加载失败时显示的文字
  • height、width指定图片宽高，指定其中一个的时候，另一个会按比例自适应，这两个是 img 的属性，不是 CSS，注意区别
  • src 图片的地址

事件

  • onload 图片加载成功的时候触发
  • onerror 图片加载失败的时候触发，图片加载失败时，可以用其他图片来代替，以提升用户体验

响应式

  • max-width:100%最大宽度为窗口的宽度，实现响应式的显示。

可替换元素

replaced element 是一种外部对象，CSS 只能控制其位置，不能控制其内容。 常见的可替换元素有iframe、video、embed、img。

table 标签

  • table 只能写三个直接子标签thead、tbody、tfoot，下一级是tr，再下一级是th、td
  • 做表格应该一行一行写
  • 表格一般写这两行去除默认的样式：border-collapse:collapse合并相邻的边框;border-spacing:0边框的间隔为零。
  • table-layout指定表格行列的大小的算法

感想

标签太多了，要多练习运用才能记得住，不然隔一段时间就会忘记，特别是 input 标签的类型很多，应该每个都写一个实例帮助记忆。
