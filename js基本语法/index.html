JS基本语法

2020.04.04

表达式和语句

  • 表达式一般都会有值，如 2 + 3 //5， 5就是该语句计算出来的值；console.log(3) //3 打印 3 是函数执行的效果，值是函数的返回值 undefined。
  • 语句可能有值也可能没有，语句一般会改变环境（声明、赋值），如 var a = 1;，该语句的值为返回值 undefined。
  • 上面并不绝对，个人理解是表达式会产生一个值，语句是执行一个行为

标识符的规则

JS中所有可以由我们自主命名的都可以称为标识符，如变量名、函数名，命名的时候要遵守命名规则:

  • 由unicode字母、$、_和数字组成，不能以数字开头，中文也是合法标识符
  • 不能是es中的关键字和保留字
  • 一般采用驼峰命名法 getImg myFunc

if else 语句

语法：

let a = 5;
if (a%2 === 0) {
    console.log("a是偶数")
}
else {
    console.log("a是奇数")
}

要点： 如果省略 {}，if只会管到条件之后的第一句。

let a = 5;
if (a > 10)
    console.log("a大于10");
    console.log("哈哈哈");

//哈哈哈

不推荐省略 {}，要简写可用 条件?成立执行语句:不成立执行语句

while和for语句

语法： while(条件){语句} 语句中一般要添加条件的增量，防止死循环
for循环被认为是while循环语法的语法糖。
？ 语法糖？听过不少，但是一直不知道是什么。。。

语法糖（Syntactic sugar），也译为糖衣语法，是由英国计算机科学家彼得·约翰·兰达（Peter J. Landin）发明的一个术语，指计算机语言中添加的某种语法，这种语法对语言的功能并没有影响，但是更方便程序员使用。通常来说使用语法糖能够增加程序的可读性，从而减少程序代码出错的机会。

语法，for遍历数组：

let arr =[];
for (let i = 0, i < arr.length, i++){
    console.log(arr[i]);
}

要点：

for (var i = 0, ,)里面setTimeout的话，把for循环完之后，再去执行setTimeout的回调函数。用let不会。

break和continue

break和continue都是作用于上一级直接父层，break退出整个父层循环，continue退出本次循环流程，开始下一次。
注意： switch要记得break，当然有时候也不使用break，来达到判断多个条件同时成立的效果。

label

语法： label1 : { console.log("this is a label."); }javascript

label 与 break、continue配合使用，主要应用在循环结构、多分支结构中，以便跳出内层嵌套体。

要点：
{ foo : 1 }javascript
在chrome会被认为是对象，在firefox会被认为是标签。

{ foo : 1; }javascript 在chrome中被认为是标签。

JS要点：

  • JS 大小写敏感
  • JS中只要不影响断句，多余的空格不会影响执行，一般为了代码可读性，符号前后添加空格
  • a && b && c 取第一个假值或者最后一个值 c，a || b || c取第一个真值或者最后一个值 c。
  • return不能回车，如果接了回车，JS会自以为是的在后面加;，返回undefined
  • 不要滥用注释，注意把坑标注
  • JS的精度问题，0.1 + 0.2 !== 0.3，到底是什么问题，我现在也说不清楚，好像是关系到进制和存储的问题，这里只做个记录。

    对于像0.1这样的数值用二进制表示你就会发现无法整除，最后算下来会是 0.000110011….由于存储空间有限，最后计算机会舍弃后面的数值，所以我们最后就只能得到一个近似值。
