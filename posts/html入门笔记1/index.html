Html入门笔记1

2020.03.14

html 是谁发明的

HTML（Hypertext Marked Language，超文本标记语言）是由 Web 的发明者 Tim Berners-Lee 和同事 Daniel W. Connolly 于 1990 年创立的一种标记语言，它是标准通用化标记语言 SGML 的应用。

html 起手式应该写什么

<!-- 文档类型 -->
<!DOCTYPE html>
<!-- html标签 -->
<html>
  <!-- head，页面信息，看不见的内容 -->
  <head lang="zh-CN">
    <!-- 字符编码 -->
    <meta charset="UTF-8" />
    <!-- 视口，内容宽度等于设备屏幕宽度，缩放比例为1 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <!-- ie浏览器使用最新的渲染模式 -->
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>
      网页标题
    </title>
  </head>
  <!-- 网页显示内容 -->
  <body>
    内容
  </body>
</html>

常用的表示章节的标签

  • h1~h6 六级标题
  • section 章节、部分
  • article 文章
  • main 主要内容
  • aside 次要内容，侧边栏
  • header 页眉，头部
  • footer 页脚，尾部
  • nav 导航栏

全局属性

  • class 类
  • id
  • hidden 隐藏元素
  • contenteditable 内容可编辑
  • style 写 css 的属性
  • tabindex 指定 tab 键选中顺序
  • title 鼠标放上去显示的文本

常用的内容标签

  • a 超链接
  • strong 重要，内容本身重要
  • em 强调，语气强调
  • code 写代码，默认使用等宽字体
  • pre 保留全部空格和回车
