JS对象的基本用法

2020.04.04

创建对象

  1. let obj = {
     "name":"minu",
     age:18
 }

  2. let obj = new Object({
     "name":"minu", 
     age:18
 })

删除对象属性

  • delete obj.age
  • obj.age = undefined 不能删除属性，只能让其属性值变成undefined

查看对象属性

  • obj.name 或者 obj["name"] 查看name的值，name是字符串
  • Object.keys(obj) 查看obj的所有key
  • Object.values(obj) 查看obj的所有值
  • Object.entries(obj) 同时查看键和值
  • console.dir(obj) 查看自身属性和原型属性
  • chrome中使用obj.__proto__或者Object.prototype查看对象的原型，第一种是chrome写法，第二种是官方写法

修改和增加对象属性

  • 修改name的值：
    obj.name = "new name"、obj["name"] = "new name"、let obj = {name: "new name"}
  • 批量添加或修改：
    Object.assign(obj, {age: 18, gender : "M})
  • 修改原型的属性：
    1. obj.toString = "new toString 只会在obj自身创建同名的toString属性，不会修改原型。
    2. object.__proto__.toString = "xxx"或者Object.prototype.toString = "xxx"会修改原型的toString方法
    3. let obj2 = {};obj.__proto__ = obj2; 会修改obj.__proto__指向为obj2，obj2.__proto__指向原型，obj.toString依然可以调用根原型的toString方法。
    4. let obj3 = Object.create(obj2) 在创建的时候指定原型。

“name” in obj 和 hasOwnProperty(“name”)的区别

  • "name" in obj 查找范围是自身以及其原型, 其中一个地方有都会返回true
  • obj.hasOwnProperty("name") 查找范围是对象自身。

要点：

  • 对象的键是字符串
  • 原型也有原型，是null，Object.prototype.__proto__
