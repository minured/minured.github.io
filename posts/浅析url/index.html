浅析URL

2020.03.26

1、URL的组成

uniform resource locator 统一资源定位符

  • 协议：规定了客户端与服务器通讯的协议
  • 域名或者ip：指定了服务器的网络位置
  • 端口：指定了服务器提供的哪种服务，默认http为80，https为443
  • 路径：指定了要访问的资源的存放路径
  • 查询参数：指定了服务器返回的内容，用于搜索、查询内容，指定页数等等
  • 锚点：本地页面某区域的定位和跳转，不包含在请求中。

2、DNS

  • Domain Name System 域名系统，主要用于域名与 IP 地址的相互转换，以及控制因特网的电子邮件的发送。
  • nslookup baidu.com 就可以查询baidu.com对应的ip地址

3、IP/ping

IP（Internet Protocol）网际互连协议，用来标记设备在网络中的位置。

ping（Packet Internet Groper），因特网包探索器，用于测试网络连接量的程序。

ping用于确定本地主机是否能与另一台主机成功交换(发送与接收)数据包，再根据返回的信息，就可以推断TCP/IP参数是否设置正确，以及运行是否正常、网络是否通畅等。

用法：ping baidu.com ，注意ping不成功也有可能是服务器拒绝ping

4、域名

Domain Name，域名通过dns映射为ip，这样就可以通过域名访问服务器，比起ip，方便记忆和传播。

域名分级：

  • 顶级域名就是一级域名，.com .cn .org等，zhihu.com是二级域名。
  • 但是，我们习惯上把qq.com称为一级域名，把pvp.qq.com称为二级域名。
