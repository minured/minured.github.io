JQuery的基本使用

2020.04.16

jQuery 如何获取元素

  1. jQuery获取元素是通过调用DOM原生的querySelectorAll()，这样就可以用css选择器来获取元素，比如：
    $('input[name=first]') 选择name=first的input元素

  2. 然后jQuery还加入了其特有的表达式，包括：

$('tr: odd') 选择表格的奇数行
$('div: gt(2)') 选择所有的div元素，除了前三个
$('div: animated') 选择当前处于动画状态的div元素

  1. jQuery加入了过滤器筛选结果，包括属性过滤、位置过滤、父子兄弟选择等

$('div').has('p') 选择包含p元素的div元素
$('div').not('.myClass') 选择class不等于myClass的div元素
$('div').filter('.myClass') 选择class等于myClass的div元素
$('div').eq(5) 选择第6个div元素

  1. jQuery选择元素的结果，默认是伪数组的形式存在，默认对结果集的全部元素应用操作

jQuery 的链式操作是怎样的

每次执行函数操作之后, 返回当前对象, 这样可以通过.继续操作当前对象, 可以将一串操作连在一起。
当操作的元素改变的时候, 返回一个新对象$(newEl)，两个对象操作的元素不会相互污染
同时jQuery还提供了一个.end()方法， 返回上一级结果集

jQuery 如何创建元素

使用template，template标签是不会被浏览器渲染的，但是它可以把html字符串变成dom节点 字符串放到template标签里之后，再通过template.content获取

let container = document.createElement("template")
container.innerHTML = "<div></div>"
let new_div = container.content.firstChild

jQuery 如何移动元素

$('div').insertAfter($('p')) 把div元素移动到p元素的后面,并返回$('div)
$('p').after($('div')) 效果同上把,把p元素放到div元素的前面, 返回$('p')

谁调用就返回谁

  • 外部, 插入兄弟元素:
  1. $移动元素.insertAfter($参照元素) \ $参照元素.after($移动元素)
  2. $p_before.insertBefore($p_after) \ $p_after.before($p_before)
  • 内部, 插入子元素:
  1. $child.appendTo($parent) \ $parent.append($child) 在内部的最后插入子元素
  2. $child.prependTo($parent) \ $parent.prepend() 在内部的前面插入子元素

jQuery 如何修改元素的属性

jQuery的获取和修改属性是用同一个函数的， 通过判断参数的个数来实现不同的操作
一般不传参数就是读取该属性的值， 默认读取结果集的第一个元素的属性值。（.text()例外，它取出所有元素的text内容）
常用的有： .html() .text() .attr() .width() .height() .val()
注意改变style使用.css()
$p.css("color") 获取color
$p.css("color", "red") 设置color

批量设置:

$p.css({
    "color": "red", 
    "fontSize": "20px"
})```
