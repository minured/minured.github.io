初识Vue

2020.05.09

Vue 的两个版本

  1. 完整版： vue.js（包含注释，开发使用） vue.mini.js(代码压缩，生产使用)
  2. 非完整版： vue.runtime.js 和 vue.runtime.mini.js

区别:
Vue 完整版，包含 compiler，会自动编译 html 模板，体积大。
runtime 版本，需要手动 render，通过 vue-loader 把 html 模板变成 js 函数 h，传入render(h){h(demo)}。优势：在 build 的时候就编译好，浏览器不需要 compiler，只要调用 h 函数，就能生成 html。

template 和 render

  • template 可以写在 vue 组件里面<template></template>， 或者作为对象的 key 传入 vue 的构造函数。 内容是 html 的模板字符串，可以使用 vue 的语法: {{n}}、@click="add"
  • render 是 runtime 版的手动渲染函数，接受一个渲染函数 h，h 的参数是一个 vue 组件的对象，通过执行 h，就可以生成 html。 
    import demo from "./demo.vue"
new Vue({
    render(h){
        h(demo)
    }
})

使用 codesandbox 写 Vue 项目

  1. 打开codesandbox.io
  2. 不要登录，只要不登录就可以一直白嫖
  3. 点击 Create a Sandbox 创建项目
  4. 选择 Vue 模板
  5. 稍等片刻，可以看到生成一个 Vue 的 demo
  6. 完成开发之后，通过左上角 file 菜单的 export to zip 下载压缩包
